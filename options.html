Hyped News Amplifier

for Hacker News and Reddit.

Hyped News Amplifier helps you easily spot news with most points and comments.

Weights: % points / % comments

If you choose 20% / 80%, then comments count 4 times as much as points. That would make messages with many comments more relevant than those with many points.

Pin / unpin messages

When selected, this option allows you to pin a message so that all the others get redistributed as if the pinned message didn't exist. Unpin to make the message count again.

Use a thermometer on Hacker News     on Reddit

When selected, this option allows you to see the hype on a thermometer rather than distributing messages. This is particularly useful on narrower themes / devices.
